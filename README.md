# Engineer Solar Wala

Engineer Solar Wala is a premier solar energy solution provider based in Lucknow, Uttar Pradesh. As a UPNEDA empaneled firm led by Er. Ashish Dixit, we specialize in high-quality solar installations, technical consultancy, and government subsidy support.

## Key Features

- **Professional Solar Installation**: Certified engineers and UPNEDA empaneled standards.
- **Expert Consultancy**: Technical energy audits and ROI analysis.
- **Subsidy Support**: Complete assistance with government schemes (PM Surya Ghar Yojana).
- **Franchise Partner Setup**: Growing network of solar professionals.
- **Staff Training**: Industry-led workshops and skill development.
- **Maintenance & AMC**: Year-round support for peak performance.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Framer Motion (Animations), Lucide React (Icons)
- **UI Components**: shadcn/ui

## Project Structure

- `src/pages/services`: Detailed service pages for different business verticals.
- `src/components`: Reusable UI components including the premium Navbar and Contact Drawer.
- `src/contexts`: Language management for Hindi/English support.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Build for production: `npm run build`
